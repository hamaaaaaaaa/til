## 3. 用語・ポイントまとめ
- **OSI参照モデル:** 7層でデータ通信を理解
- **TCP/IP:** 信頼性のある通信を提供
- **公開鍵暗号/共通鍵暗号:** 暗号化手法の種類
- **ファイアウォール:** 不正アクセス防止の仕組み

---

## 4. 例題・問題演習
- 問題: TCP/IPの4層モデルの各層の役割は？
- 解答:
  1. アプリケーション層: ソフト間通信
  2. トランスポート層: データ送受信の管理
  3. ネットワーク層: 経路制御
  4. データリンク層: 物理的伝送制御
